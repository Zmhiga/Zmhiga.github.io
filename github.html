abc.mse R Package

In this project, I created an R package to utilize approximate Bayesian computation (ABC) to find the posterior distribution of parameter(s) for the exponential distribution, gumbel distribution, and Cauchy distribution given the true parameter values found from the prior distribution. Using a generating mechanism to obtain the posterior distribution of the parameter(s), we then find the mean-squared error of the given parameter(s). The R package can be installed via Github.
